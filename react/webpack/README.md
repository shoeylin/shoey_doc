官網下載npm 安裝npm 

查看npm版本 

> npm -v

安裝webpack

> npm install -g webpack

查看webpack版本

> webpack -v

> npm install -g webpack-cli
全局安裝會只能用特定版本

創建一個資料夾

> mkdir hello-webpack

進入資料夾

初始化npm

> npm init

安裝依賴

> npm install --save-dev webpack

創建目錄

> mkdir src

> touch src/index.js

webpack

webpack --mode=development
